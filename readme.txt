This applications function is to add tasks to a list
each of which containing a description and a due date

The steps to use the application summarized are:
Add a task using the add task feature with decription and due date
display the task list to see tasks added with ID
Tasks will be shown on the left textbox
You may edit or remove tasks as needed by typing the task number into the Task Num box
then you may edit task by typing in the New boxes
or remove aid task by pressing the remove task button
You may remove all tasks using the clear button at the bottom
