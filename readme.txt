This applications function is to add tasks to a list
each of which containing a description and a due date

The steps to use the application summarized are:

1) Add a task using the add task feature with description and due date

2) Display the task list to see tasks added with ID. Tasks will be shown on the left textbox

4) You may edit or remove tasks as needed by typing the task number into the "Task Num" box,
then you may edit task by typing in the New boxes

5) You may also remove a task by typing the task ID in THE SAME "Task Num" textbox and clicking remove Task

6) You may remove all tasks using the clear button at the bottom
